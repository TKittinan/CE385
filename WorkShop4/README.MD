3.1.js
เป็นฟังก์ชันคำนวณพื้นฐาน เช่น บวก ลบ คูณ และหาร ใช้เป็นตัวอย่างของฟังก์ชันแบบ synchronous และการ export module เพื่อนำไปใช้งานในไฟล์อื่น

3.2.js
แสดงตัวอย่างการทำงานของ asynchronous แบบ Callback และ Promise เพื่อเปรียบเทียบความแตกต่างของทั้งสองรูปแบบ โดย Callback จะเรียกใช้งานหลังจากการทำงานเสร็จ ส่วน Promise จะใช้ .then() และ .catch() ในการจัดการผลลัพธ์และข้อผิดพลาด

3.3.js
เป็นตัวอย่างการใช้งาน async/await เพื่อให้โค้ดอ่านง่ายขึ้น และใช้ try...catch ในการจัดการ error แสดงให้เห็นว่าการใช้ async/await สามารถทำงานแทน Promise แบบเดิมได้

3.4.js
เป็นการจำลองการดึงข้อมูลจากหลายเซิร์ฟเวอร์พร้อมกัน โดยใช้
Promise.race เพื่อเลือกผลลัพธ์จาก Promise ที่เสร็จเป็นตัวแรก
Promise.allSettled เพื่อแสดงผลลัพธ์ของ Promise ทุกตัว ไม่ว่าจะสำเร็จหรือเกิดข้อผิดพลาด
