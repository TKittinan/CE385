WorkShop1.1.js: การประกาศตัวแปรพื้นฐานเพื่อจัดเก็บข้อมูลส่วนบุคคล เช่น ชื่อ อายุ และสาขาวิชา แล้วนำมาแสดงผลผ่านคอนโซล

WorkShop1.2.js: การใช้งานตัวดำเนินการ typeof เพื่อตรวจสอบและแสดงประเภทของข้อมูลชนิดต่างๆ เช่น String, Number, Boolean, Null และ Undefined

WorkShop1.3.js: การคำนวณคะแนนรวมและค่าเฉลี่ยจากตัวเลข 3 ชุด พร้อมใช้ตัวดำเนินการเปรียบเทียบเพื่อตรวจสอบว่าผลลัพธ์ผ่านเกณฑ์ที่กำหนดหรือไม่

WorkShop1.4.js: การสร้างระบบตรวจสอบสิทธิ์โดยใช้เงื่อนไข if-else เพื่อเช็คความถูกต้องของ Username, Password และตรวจสอบเกณฑ์อายุของผู้ใช้งาน
